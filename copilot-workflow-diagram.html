Error Investigation Workflow

Operational error triage — manual process vs. GitHub Copilot assisted

Current — Manual Process
🚨 Operational Error Reported
↓
Step 1
Open ELK / Kibana Dashboard
Navigate to the correct index, set time range, search for the service and error level.
⏱ Manual navigation
↓
Step 2
Locate & Extract Error Logs
Filter by traceId / correlationId, copy relevant stack traces and log lines manually.
⏱ Copy-paste effort
↓
Step 3
Open Source Code in IDE
Switch to the codebase, navigate the relevant service, controller, repository files manually.
⏱ Context switching
↓
Step 4
Correlate Log ↔ Source Manually
Read the stack trace line by line, find the matching class and method, reason about the failure path.
⏱ Deep mental effort
↓
Step 5
Identify Root Cause
Piece together the error after fully reading multiple files — often repeated across incidents.
⏱ Slow & error-prone
↓
😓 Root cause found — eventually
30–90 min per incident depending on engineer familiarity
VS
Target — GitHub Copilot Assisted
🚨 Operational Error Reported
↓
Step 1
Fetch Error Log from ELK
Copy the raw error log / stack trace from Kibana as before — this remains the human input.
↓
Step 2 — GitHub Copilot Chat
Add Source Code Context
Attach multiple source subfolders as context using #folder or #file references in Copilot Chat.
📁 #folder controller/
📁 #folder service/
📁 #folder document/
📁 #folder exception/
📋 Paste log Raw ELK error + stack trace
✓ Single context window
↓
Step 3 — Copilot Analysis
Automatic Log ↔ Source Correlation
Copilot reads all provided files, maps the exception class and line numbers in the stack trace to the exact source locations, and traces the failure path.
✓ No manual file navigation
↓
Step 4 — Copilot Output
Root Cause & Fix Suggestion
Copilot identifies the exact method, condition, and likely fix — with code references and explanation.
✓ Actionable diagnosis
↓
⚡ Root cause identified instantly
Minutes per incident, consistent across all engineers regardless of familiarity
🕐
Manual investigation — typical time
30 – 90 minutes
→
⚡
Copilot-assisted investigation — typical time
2 – 5 minutes
